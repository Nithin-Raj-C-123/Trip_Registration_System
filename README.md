# Trip Registration System

A simple web application developed using PHP and MySQL that allows users to register for trips through an online registration form. The submitted information is stored in a MySQL database, making it suitable for managing participant registrations efficiently.

---

## Features

- User Registration Form
- PHP Form Handling
- MySQL Database Integration
- Dynamic Success Message
- Responsive User Interface
- Local Development using XAMPP

## Installation

1. Install XAMPP.
2. Start Apache and MySQL.
3. Create a database named `trip`.
4. Create the required table in phpMyAdmin.
5. Place the project inside:

```
C:\xampp\htdocs\
```

6. Open your browser and visit:

```
http://localhost/TripRegistration/
```

---

## Functionalities

- Accepts participant details through an HTML form.
- Validates form submission using the POST method.
- Inserts records into the MySQL database.
- Displays a confirmation message after successful registration.
- Handles database connection using MySQLi.
